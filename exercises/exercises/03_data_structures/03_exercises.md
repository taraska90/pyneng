Из списка нужно получить уникальный список VLANов, отсортированный по возрастанию номеров.

Не использовать для решения задачи циклы, оператор if.

```python
VLANS = [10, 20, 30, 1, 2, 100, 10, 30, 3, 4, 10]
```
